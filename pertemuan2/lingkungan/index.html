Praktikum Dimulai

  1. Buka Launcher Menu dengan tekan tombol Super (logo Windows).
  • Buka Aplikasi yang ada di Ubuntu
  1. Mengubah Background.
  • Klik ikon System Settings pada Launcher untuk menjalankan program setup.
  • Pada bagian Personal pilih Appearance
  • Ubahlah background dengan memilih gambar
  1. Mengubah Themes.
  • Klik ikon System Settings pada Launcher untuk menjalankan program setup.
  • Pada bagian Personal pilih Appearance.
  • Ubahlah themes dengan memilih theme lain pada menu dropdown.
  1. Install aplikasi melalui Terminal. sudo apt-get install {nama aplikasi}
  2. Install aplikasi melalui Ubuntu Software Center.
