Pemrosesan File Teks.

First Things First

  1. Angkat tangan jika penjelasan terlalu cepat
  2. Mari membuat suasana praktikkum yang kondusif
  3. Aktiflah bertanya agar tidak bingung di kemudian hari

Sebelum dimulai…

  1. Ayo baca dulu sebentar
  2. Beberapa tool atau perintah yang dapat digunakan untuk memanipulasi file teks :
    • cat ­- concatenate
    • more dan less
    • head dan tail
    • wc – word count
    • cut
    • sort
    • tr ­ - translate
    • sed – stream editor
    • grep – global regular expression print
    • find

Praktikum dimulai…

  1. Menggunakan perintah cat

    • $ cat > latihan
      Saya sedang belajar Linux
      {Ctrl}+D
    • $ cat latihan
    • $ cat >> latihan
      Distro Linux yang dipelajari adalah ubuntu
      {Ctrl}+D

  2. Menggunakan perintah more dan less

    • $ more /etc/passwd
    • $ ls -l /etc | more
    • $ less /etc/passwd
    • $ ls -l /etc | less

  3. Menggunakan perintah head dan tail

    • $ head /etc/passwd
    • $ head -n5 /etc/passwd
    • $ tail /etc/passwd
    • $ tail -n5 /etc/passwd
    • $ tail -n+5 /etc/passwd

  4. Menggunakan perintah wc

    • $ wc /etc/passwd
    • $ wc -l /etc/passwd
    • $ wc -w /etc/passwd
    • $ wc -c /etc/passwd

  5. Menggunakan perintah cut

    • $ cut -d":" -f1-4 /etc/passwd
    • $ cut -d":" -f1,4 /etc/passwd
    • $ cut -c1-4 /etc/passwd

  6. Menggunakan perintah sort

    • $ sort /etc/passwd
    • $ sort -t":" -k3 /etc/passwd
    • $ sort -t":" -k3 -n /etc/passwd
    • $ sort -t":" -k3 -nr /etc/passwd

  7. Menggunakan perintah tr

    • $ tr abc ABC < /etc/passwd
    • $ tr a-z A-Z < /etc/passwd
    • $ tr -d aiueo < /etc/passwd

  8. Menggunakan perintah sed

    • $ sed -e "s/root/badu/g" /etc/passwd
    • $ echo "s/root/badu/g" > skrip
    • $ sed -f skrip /etc/passwd

  9. Menggunakan perintah grep

  • $ grep root /etc/passwd
  • $ grep echo *
  1. Menggunakan perintah find
    • $ find . -name "latih*"
    • $ find . -name "latih*" -perm 644
    • $ find . -name "latih*" -perm 644 -user dudi
    • $ find . -name "latih*" -perm 644 -exec rm -i {} \;

Selesai.
