Praktikum dimulai.

Login dan Logout via GUI.

  1. Login via GUI .

    • Masukan username dan password.

  2. Logout via GUI.

    • Klik Logout.

Login dan Logout via Terminal.

  1. Login via Terminal.

    • Tekan ctrl+alt+f1 - ctrl+alt+f6
    • Masukan username dan password.

  2. Logout via Terminal.

    • tekan # exit atau # logout.

Perintah Dasar Melalui Terminal.

  1. Membuka Terminal.

    • ctrl+alt+t

  2. Berubah dari user biasa ke root.

    • $ sudo su
      Kembali ke user biasa dengan tekan # exit

  3. Mengubah Password.

    • Untuk mengubah password user biasa, $ passwd
    • Untuk mengubah password user root, $ sudo passwd root

  4. Shutdown Komputer.

    • sudo poweroff atau sudo shutdown -h now

  5. Restart Komputer.

    • sudo reboot atau sudo shutdown -r now
