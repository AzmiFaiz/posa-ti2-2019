Instalasi Linux dengan Metode Dual Boot.

Sebelum Dimulai…

  • Cari cara untuk masuk BIOS / Recovery Mode di komputer masing-masing
  • Nyalakan dengan sim ejector / peniti khusus untuk yang memerlukan
  • Matikan Opsi Fast Startup dari BIOS

Proses Instalasi Ubuntu.

  1. Nyalakan komputer dan Masuk ke BIOS Mode/Recovery Mode.
  2. Pilihlah untuk melakukan Boot dari USB, kebanyakan komputer menekan tombol F12 sebelum masuk ke logo Windows, tombol F2 & F10 juga menjadi alternatif
  3. Akan muncul pilihan yang berbeda saat mulai install ubuntu. Pilih Install Ubuntu

  1. Jangan disambungkan ke jaringan internet, agar proses instalasi menjadi lebih cepat karena tidak update
  2. Mulai muncul pilihan bahasa, Pilih bahasa Indonesia jika lebih nyaman menggunakan Bahasa Indonesia. Klik Continue
  3. Mulai muncul pilihan keyboard Layout. Jika tidak memakai keyboard dengan input khusus maka bisa di klik Continue
  4. Pada bagian updates and other software pilih/ceklis normal installation dan install third party…
    Untuk pilihan Download Updates bisa di uncheck
  5. Di bagian Installation type pilihlah Something else

Perhatikan langkah selanjutnya karena kalau salah pilih partisi maka data kita pun akan hilang :(

  1. Pilih partisi yang sudah dibuat sebelumnya. Diingat kembali besar partisi yang dibuat.

  2. Pilih Partisi yag sudah ditentukan menjadi mount point /

  3. Pilih partisi dengan size 3GB sebagai mount point swap

  4. Setelah konfigurasi penyimpanan akan muncul warning dari ubuntu. pastikan partisi yang dipiliih benar dan akan di format

  5. Pilih lokasi kita berada. Pilih Jakarta

  1. Isi login detail dengan username dan password yang dikehendaki

  1. Proses Instalasi akan berjalan

  2. Jika proses instalasi berjalan dengan baik maka akan muncul pilihan restart

Selesai.
