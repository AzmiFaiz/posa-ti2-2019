  1. Angkat tangan jika penjelasan terlalu cepat
  2. Mari membuat suasana praktikkum yang kondusif
  3. Aktiflah bertanya agar tidak bingung di kemudian hari

Sebelum dimulai…

Alat yang diperlukan

  1. Siapkan FlashDisk kosong berkapasitas min. 8GB
  2. Install software PowerISO
  3. Install Software Minitool Partition Wizard
  4. Ubuntu 18.04 ISO
  5. Siapkan peniti/sim ejector untuk laptop tertentu masuk recovery mode
  6. Pastikan Baterai Laptop diatas 50% / Terhubung dengan power adapter

Siap-siap dulu yuk…

Matikan opsi fast startup di Windows 8 / 10.

  1. Klik kanan pada ikon battery lalu pilih Power Options

  2. Klik “Choose what the power buttons do”

  3. Scroll down ke “Shutdown settings” dan uncheck pada tulisan “Turn on fast startup”.
    Note: Jika tulisan kotak ini belum bisa di uncheck maka kita harus klik “Change settings that are currently unavailable”.

  4. Klik “Save Changes”

Membuat Partisi Untuk Linux dan Swap.

  1. Buka aplikasi MiniTool Partition Wizard, Pilih Launch Application.
    Pilih partisi yang akan di split untuk penyimpanan Linux dan Swap. Lalu klik split partition -> split

  2. Drag Slider nya untuk menentukan besarnya penyimpanan Linux. Klik OK jika sudah selesai. Note : Rekomendasi dari saya, 25GB untuk Linux & 3GB untuk Swap

  3. Kembali ke tampilan awal dan klik Apply untuk membuat partisi yang diinginkan. Tunggu Prosesnya.

Membuat bootable Flash Disk sebagai sumber instalasi Linux.

  1. Buka Aplikasi PowerISO, Pastikan FD sudah di backup & sudah terbaca oleh komputer. Klik Tools -> “Create bootable USB Drive”

Klik OK

  1. Klik gambar folder pada pilihan Image File, cari letak file ISO Ubuntu, lalu pilih. Pastikan Image File berisi lokasi dari file ISO Ubuntu

  2. Untuk Destination USB Drive pastikan kita telah memilih FlashDisk yang sudah disiapkan

  3. Klik Start untuk memulai proses pembuatan bootable FlashDisk.

  4. Akan muncul peringatan dari PowerISO, pastikan kembali bahwa yang muncul adalah FD yang sudah disiapkan.

  • PowerISO akan mulai membuat Bootable USB, dan menunjukkan progress & estimasi waktu penulisan data. Nantinya akan muncul popup message “Writing USB drive completed successfully.” setelah prosesnya selesai.
  1. Jika tidak muncul pesan error, maka proses yang dilakukan telah berhasil. Maka kita sudah siap untuk Install Linux
