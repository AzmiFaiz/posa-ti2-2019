Shell - Command - Vim

First Things First

  1. Angkat tangan jika penjelasan terlalu cepat
  2. Mari membuat suasana praktikkum yang kondusif
  3. Aktiflah bertanya agar tidak bingung di kemudian hari

Sebelum dimulai…

  1. Apa itu Shell.

  2. Jenis-jenis Shell: Korn, C, Bourne, BASH (Bourne-Again Shell).

  3. Command / Perintah.

  4. Perintah Dasar:

    • ls: melihat isi folder.
    • cd: pindah folder / direktori.
    • pwd: mengecek posisi sekarang.
    • mkdir: membuat folder.
    • touch: membuat file.
    • cp: mengcopi file / folder.
    • mv: memindahkan file / folder.
    • cat: membaca file.

  5. Apa itu VIM?

Praktikum dimulai

Shell.

  1. Cek Shell: $ echo $0

Perintah Dasar.

Sintak penulisan command: command-name [-options] <argument>

Melihat isi Folder / Direktori: ls

  1. ls
  2. ls -l
  3. ls -l Download

Cek Posisi Sekarang: pwd

  1. pwd

Berpindah Folder / Direktori: cd

  1. cd Download
  2. cd ../
  3. cd Desktop

Membuat Folder: mkdir

  1. mkdir Praktikum
  2. mkdir Tugas Kuliah
  3. mkdir "Tugas POSA"

Membuat File: touch

  1. touch Latihan
  2. touch Belajar POSA
  3. touch "Belajar POSA"

Menghapus file / folder: rm / rmdir

  1. touch satu
  2. rm satu
  3. mkdir dua
  4. rm dua
  5. mkdir tiga
  6. rmdir tiga
  7. mkdir empat
  8. touch empat/file
  9. mkdir empat
  10. rm -r empat

Mengcopy File / Folder: cp

  1. touch file
  2. cp file filedua
  3. cp filedua filetiga
  4. mkdir folder
  5. cp folder folderdua
  6. cp folderdua foldertiga

Command History

  1. tekan tombol atas atau bawah
  2. history

Teks Editor VIM.

Membuka VIM

  1. vim.tiny atau vim.tiny nama-file

Mode VIM

  1. Editor: Mode untuk menjalankan perintah.
  2. Insert: Mode untuk menulis.

Menulis di VIM

Masuk ke Mode Insert untuk menulis

  1. tekan a atau i
  2. Tulis teks berikut: Sedang Berada di Mode Insert

Menyimpan File

Untuk menyimpan file gunakan perintah :wq.

Masuk ke Mode Editor untuk menjalankan perintah

  1. tekan escape (esc)
  2. tekan :wq

Selesai.
