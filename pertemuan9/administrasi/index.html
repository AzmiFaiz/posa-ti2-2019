Administrasi Sistem dan Pengguna

First Things First

  1. Angkat tangan jika penjelasan terlalu cepat
  2. Mari membuat suasana praktikkum yang kondusif
  3. Aktiflah bertanya agar tidak bingung di kemudian hari

Administrasi Sistem

Berubah User menjadi Root
  1. Berubah menjadi user root: sudo su
  2. Keluar dari user root: exit
Manajemen Proses
  1. Menampilkan proses: ps atau ps -A atau ps -ef
  2. Menampilkan struktur proses: pstree
  3. Menampilkan proses real time: top
  4. Melihat daftar sinyal: kill -l
  5. Mengirimkan sinyal kill: kill -9 no-pid
Monitoring Sistem
  1. Menampikan penggunaan file system: df atau df -h
  2. Melihat ukuran direktori: du atau du -h
  3. Melihat ukuran direktori Music: du -h Music
  4. Melihat penggunaan memori: free atau free -h

Administrasi User di Terminal

Membuat User
  1. Membuat user fikri dan membuat home directory: sudo useradd -m fikri
  2. Cek user fikri di file passwd: grep fikri /etc/passwd
  3. Melihat direktori home fikri: ls -l home
Menghapus User
  1. Menghapus user fikri dan home directory: sudo userdel -r fikri
  2. Cek user fikri di file passwd: grep lilik /etc/passwd
  3. Cek home direktori fikri: ls -l home
Membuat Grup
  1. Membuat grup ti1 = sudo groupadd ti1
  2. Cek group ti1 di file /etc/group: grep ti1 /etc/group
Modifikasi Grup
  1. Mengubah grup ti1 menjadi ti2: sudo groupmod -n ti2 ti1
  2. Cek group ti1 di file /etc/group: grep ti1 /etc/group
  3. Cek group ti2 di file /etc/group: grep ti2 /etc/group
Menghapus Grup
  1. Menghapus grup ti2: sudo groupdel ti2
  2. Cek group ti2 di file /etc/group: grep ti2 /etc/group
Membuat Password User
  1. Membuat user hilmi: sudo useradd -m hilmi
  2. Cek password hilmi di /etc/shadow: sudo grep hilmi /etc/shadow
  3. Membuat password hilmi: sudo passwd hilmi
  4. Cek password hilmi di /etc/shadow: sudo grep hilmi /etc/shadow
Membuat Password Grup
  1. Membuat grup ti1: sudo groupadd ti1
  2. Cek password ti1 di /etc/gshadow: sudo grep ti1 /etc/gshadow
  3. Membuat password grup ti1: sudo gpasswd ti1
  4. Cek password ti1 di /etc/gshadow: sudo grep ti1 /etc/gshadow
Menambahkan User ke Grup
  1. Menambahkan user hilmi ke grup ti2: sudo gpasswd -a hilmi ti2
  2. Cek user hilmi di grup ti2: sudo grep ti2 /etc/gshadow
